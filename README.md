# Flask Todo Application

This is a simple Todo application built using Flask and SQLAlchemy. The application allows users to add, update, delete, and edit todo tasks.

## Features

- Add new todo tasks
- Mark tasks as complete or incomplete
- Edit existing tasks
- Delete tasks

## Requirements

- Python 3.x
- Flask
- Flask-SQLAlchemy
